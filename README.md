# 履歴書ビルダー

自由に項目をカスタマイズできる、Web ベースの履歴書作成アプリケーションです。  
ES（エントリーシート）提出でフォーマット指定がない企業向けに、整った A4 履歴書を作成・PDF 出力できます。

## 特徴

- **項目の自由なカスタマイズ** — セクションの追加・削除・並び替え・タイトル変更に対応
- **2種類の入力タイプ** — テキスト（自由記述）/ 年月＋内容（経歴向け）
- **リアルタイムプレビュー** — 左パネルで入力、右パネルで A4 履歴書を即時確認
- **A4 ページ分割表示** — プレビュー上でページ境界を視覚的に表示
- **改ページ制御** — `break-inside: avoid` によりセクション途中での不自然な改ページを防止
- **文字数カウント** — 各テキスト入力欄にリアルタイム文字数表示
- **ハイパーリンク対応 PDF** — テキスト中の URL・メールアドレスを自動検出し、PDF 出力時にクリック可能なリンクとして保持
- **証明写真アップロード** — 画像ファイルを選択して履歴書に反映

## セットアップ

```bash
git clone https://github.com/<your-username>/resume-builder.git
cd resume-builder
npm install
npm run dev
```

ブラウザで `http://localhost:5173` を開いてください。

## ビルド

```bash
npm run build
```

`dist/` に静的ファイルが生成されます。GitHub Pages、Vercel、Netlify 等にデプロイ可能です。

## 技術スタック

- [React](https://react.dev/) 18
- [Vite](https://vite.dev/) 6
- CSS-in-JS（ライブラリ不使用、インラインスタイル）

## 使い方

1. 左パネルで各セクションの情報を入力する
2. 右パネルでリアルタイムにプレビューを確認する
3. 「＋ セクションを追加」から任意の項目を追加する
4. 矢印ボタンで順番を入れ替え、セクション名はダブルクリックで変更可能
5. 完成したら右上の「印刷 / PDF保存」ボタンで出力する

PDF 保存時は、印刷ダイアログで「PDF として保存」を選択してください。  
テキスト内に記載した URL やメールアドレスは、PDF 上でクリック可能なリンクとして機能します。
